\# ☕ Spill The Tea



\*\*Spill The Tea\*\* is a console-based cafe chain management system built in \*\*C++ using Object-Oriented Programming\*\*. Set around a fictional Pakistani chai brand, the project manages 100+ branches and analyzes their sales, costs, and customer trends.



The name \*Spill The Tea\* is a little play on words, besides the obvious chai theme, "spilling the tea" also means sharing gossip, which fits the idea of uncovering insights and "telling the story" hidden in the data.



\## Features



\*  Manage 100+ cafe branches across Pakistan

\*  Track 24 months of sales, costs, and customer data

\*  Generate sales forecasts from previous trends

\*  Rank branches based on overall performance

\*  Group branches using a custom K-Means clustering algorithm

\*  Display analytics with ASCII charts in the terminal

\*  Save and load data from text files



\## Concepts Used



\* Multi-level inheritance 

\* Composition and aggregation

\* Dynamic memory management

\* Rule of Three

\* Constructor chaining

\* Operator overloading

\* File handling

\* Basic analytics and clustering algorithms



Built as a university OOP project, \*\*Spill The Tea\*\* was my way of exploring advanced C++ concepts while creating something fun and slightly tongue-in-cheek—a cafe management system that doesn't just serve tea, but also \*spills it\*. 😉
